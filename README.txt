# USX 単体運転記録表アプリ

## フォルダ構成
```
📁 USX運転記録表アプリ/
  ├ 📄 USX運転記録表を起動.bat      ← Windowsはこれをダブルクリック
  ├ 📄 USX運転記録表を起動.command  ← Macはこれをダブルクリック
  ├ 📄 server.py                    ← サーバー本体
  ├ 📄 template.xlsx                ← Excelテンプレート（変更しないこと）
  └ 📁 static/
      └ 📄 index.html               ← 入力画面
```

## 使い方

### PC（初回のみ）
1. Python 3 をインストール → https://www.python.org/downloads/
   - Windowsは「Add Python to PATH」にチェックを入れること

### 現場での入力（スマホ・タブレット）
1. PCとスマホを同じWi-Fiに接続
2. PCで起動バッチを実行
3. コンソールに表示される `http://192.168.x.x:5199` をスマホのブラウザで開く
4. 各タブで数値を入力

### Excel出力（PC）
1. 起動バッチをダブルクリック → ブラウザが自動で開く
2. 入力済みデータを確認
3. 「Excel出力」タブ → 📥 Excel出力 ボタンを押す
4. デスクトップの「USX運転記録」フォルダに保存されます

## 注意事項
- サーバーを終了するにはコンソールウィンドウを閉じてください
- template.xlsx は書き換えないでください
- 出力ファイルは毎回タイムスタンプ付きで保存されます
